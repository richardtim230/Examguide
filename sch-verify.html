Get help with uploading and managing your school data

How to Upload Student Data?

1. Go to the "Upload Data" section
2. Select the appropriate year and term
3. Choose the CSV or Excel file to upload
4. Click "Upload" to submit
5. Monitor the upload status in "Upload History"

File Format Requirements

• Supported Formats: CSV, XLSX, XLS
• Maximum File Size: 10MB
• Character Encoding: UTF-8 recommended
• Required Columns: Check the downloadable template

Common Issues & Solutions

Upload Failed? Check file format and size
Missing Data? Verify all required columns are present
Still Having Issues? Contact support: support@examguard.com

Data Security

• All data is encrypted during transmission
• Data is securely stored on ExamGuard servers
• Only authorized personnel can access your data
• Regular security audits and backups performed
