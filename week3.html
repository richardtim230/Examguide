Day 1 (Thursday): Floats & Basic Positioning

What's Happening Today?

Today, you'll learn how to arrange elements side by side using floats and control their position on the page with CSS positioning properties. These are foundational techniques for creating layouts before moving to more modern methods.

Concepts – Pay Attention to This:

Floats:

  • float: Moves an element to the left or right, allowing content to wrap around it (originally designed for text wrapping around images).
  • Values: left, right, none.
  • Example: float: left; makes an element move to the left, with other content wrapping around its right side.
  • Clearing Floats: When elements are floated, they can disrupt the normal flow of the page. Use clear to ensure elements appear below floated elements.
    • clear: left;, clear: right;, clear: both;.
    • Clearfix Hack: Used to ensure a parent container wraps around its floated children.
  • Example (Float with Clearfix):
    • .container {
    overflow: auto; /* Clearfix method */
}
.float-left {
    float: left;
    width: 50%;
}
.float-right {
    float: right;
    width: 50%;
}
  • Note: Floats are older and less flexible than Flexbox or Grid, but understanding them is useful for legacy code or specific use cases like image alignment.

CSS Positioning:

  • position: Defines how an element is positioned in the document.
    • static: Default. Element follows normal flow.
    • relative: Positioned relative to its normal position. Use top, right, bottom, left to offset.
    • absolute: Positioned relative to the nearest positioned ancestor (not static). Removes element from normal flow.
    • fixed: Positioned relative to the viewport. Stays in place even when scrolling.
    • sticky: Toggles between relative and fixed based on scroll position.
  • z-index: Controls stacking order of positioned elements (higher values appear on top).
  • Example:
    • .relative-box {
    position: relative;
    top: 10px;
    left: 20px;
}
.fixed-button {
    position: fixed;
    bottom: 20px;
    right: 20px;
    z-index: 100;
}

Your Hands-On Activity: Create a Two-Column Layout & Fixed Button

Create a Two-Column Layout with Floats:

  • Open your index.html or create a new layout.html.
  • Add two <div> elements for content (e.g., main content and sidebar).
    • <div class="container">
    <div class="main-content">
        <h2>Main Content</h2>
        <p>This is the main content area...</p>
    </div>
    <div class="sidebar">
        <h2>Sidebar</h2>
        <p>This is the sidebar content...</p>
    </div>
</div>
  • In styles.css, style the layout with floats:
    • .container {
    overflow: auto; /* Clearfix */
}
.main-content {
    float: left;
    width: 70%;
    padding: 15px;
}
.sidebar {
    float: right;
    width: 30%;
    padding: 15px;
    background-color: #f0f3f5;
}
  • Add a footer below the floated elements using clear: both;:
    • <footer>
    <p>Footer content goes here.</p>
</footer>
    footer {
    clear: both;
    text-align: center;
    padding: 10px;
}

Add a "Back to Top" Button:

  • Add a button to your HTML:
    • <button class="back-to-top">Back to Top</button>
  • Style it with position: fixed:
    • .back-to-top {
    position: fixed;
    bottom: 20px;
    right: 20px;
    padding: 10px 20px;
    background-color: #2980b9;
    color: white;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    z-index: 100;
}
  • Note: The button won't scroll with the page due to position: fixed. Test it by adding enough content to your page to enable scrolling.

Your Homework:

  • Recreate a Magazine Layout: Create a new HTML file (e.g., magazine.html) and CSS file.
  • Design a simple magazine-style layout with:
    • A floated image on the left or right with text wrapping around it.
    • A main article section and a sidebar with related content.
    • Use clear or clearfix to ensure proper layout.
  • Apply positioning to place a decorative element (e.g., a badge or icon) using position: absolute within a relatively positioned container.

Day 2 (Friday): Flexbox - 1D Layout

What's Happening Today?

You'll learn Flexbox, a modern CSS layout model for arranging elements in a single dimension (row or column). It's powerful for creating responsive layouts with less hassle than floats.

Concepts – Pay Attention to This:

Flexbox Intro:

  • Flexbox is a one-dimensional layout system, ideal for rows or columns.
  • It consists of a flex container (parent) and flex items (children).
  • Apply display: flex; to the parent to enable Flexbox.

Flex Container Properties:

  • display: flex;: Turns an element into a flex container.
  • flex-direction: Defines the main axis (row, row-reverse, column, column-reverse).
  • justify-content: Aligns items along the main axis (flex-start, flex-end, center, space-between, space-around, space-evenly).
  • align-items: Aligns items along the cross axis (flex-start, flex-end, center, baseline, stretch).
  • flex-wrap: Controls whether items wrap to new lines (nowrap, wrap, wrap-reverse).
  • align-content: Aligns wrapped lines when there's extra space (flex-start, flex-end, center, space-between, space-around, space-evenly).
  • Example:
    • .flex-container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
    flex-wrap: wrap;
}

Flex Item Properties:

  • order: Changes the order of a flex item (default is 0).
  • flex-grow: Defines how much an item grows relative to others (e.g., 1).
  • flex-shrink: Defines how much an item shrinks relative to others.
  • flex-basis: Sets the initial size of an item before growing/shrinking (e.g., 200px).
  • flex: Shorthand for flex-grow, flex-shrink, flex-basis (e.g., flex: 1 1 200px;).
  • align-self: Overrides align-items for a specific item.
  • Example:
    • .flex-item {
    flex: 1;
    align-self: flex-end;
}

Your Hands-On Activity: Convert to Flexbox & Create a Navigation Bar

Convert Float Layout to Flexbox:

  • Open your layout.html or index.html from Day 1.
  • Replace the float-based layout with Flexbox:
    • .container {
    display: flex;
    flex-direction: row;
    gap: 20px;
}
.main-content {
    flex: 2; /* Takes up twice as much space as sidebar */
    padding: 15px;
}
.sidebar {
    flex: 1;
    padding: 15px;
    background-color: #f0f3f5;
}
  • Remove float and clear properties.
  • Test with Live Server and resize the browser to see how Flexbox handles wrapping (if flex-wrap: wrap; is set).

Create a Responsive Navigation Bar:

  • In your HTML, add a <nav> element with a list of links:
    • <nav class="navbar">
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
  • Style it with Flexbox:
    • .navbar ul {
    display: flex;
    justify-content: space-evenly;
    list-style: none;
    padding: 0;
}
.navbar li {
    padding: 10px;
}
.navbar a {
    text-decoration: none;
    color: #2980b9;
}
  • Add hover effects (you'll learn more about these next week):
    • .navbar a:hover {
    color: #1f6391;
    text-decoration: underline;
}

Your Homework:

  • Build an Image Gallery or Product Cards: Create a new HTML file (e.g., gallery.html) and CSS file.
  • Add several images or product cards (use <div> with an <img> and text).
  • Use Flexbox to arrange them in a row, with wrapping for responsiveness:
    • .gallery {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    justify-content: center;
}
.gallery-item {
    flex: 1 1 200px; /* Grow, shrink, basis */
    text-align: center;
}
  • Test responsiveness by resizing your browser.

Day 3 (Saturday): CSS Grid - 2D Layout

What's Happening Today?

You'll learn CSS Grid, a powerful two-dimensional layout system for creating complex page layouts. Grid is ideal for structuring entire pages or sections with precise control over rows and columns.

Concepts – Pay Attention to This:

CSS Grid Intro:

  • Grid is a two-dimensional layout system, controlling both rows and columns simultaneously.
  • Like Flexbox, it involves a grid container and grid items.
  • Apply display: grid; to the parent to enable Grid.

Grid Container Properties:

  • display: grid;: Turns an element into a grid container.
  • grid-template-columns: Defines the number and size of columns (e.g., 100px 200px, 1fr 2fr).
  • grid-template-rows: Defines the number and size of rows.
  • gap: Sets spacing between grid cells (e.g., gap: 20px;).
  • fr Unit: Represents a fraction of available space (e.g., 1fr 1fr splits space equally).
  • repeat(): Simplifies defining multiple columns/rows (e.g., repeat(3, 1fr) for three equal columns).
  • minmax(): Sets a range for sizes (e.g., minmax(100px, 1fr)).
  • Example:
    • .grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 100px 200px;
    gap: 15px;
}

Grid Item Properties:

  • grid-column: Specifies which column(s) an item spans (e.g., grid-column: 1 / 3; spans from column 1 to 3).
  • grid-row: Specifies which row(s) an item spans.
  • Example:
    • .grid-item {
    grid-column: 1 / span 2; /* Spans two columns */
    grid-row: 1 / 2;
}

Your Hands-On Activity: Create a Grid Layout

Create a Dashboard or Blog Post Layout:

  • Create a new HTML file (e.g., dashboard.html) and CSS file.
  • Add semantic HTML for a dashboard or blog layout:
    • <div class="grid-container">
    <header>Site Header</header>
    <nav>Navigation Menu</nav>
    <main>Main Content</main>
    <aside>Sidebar</aside>
    <footer>Footer</footer>
</div>
  • Style with CSS Grid:
    • .grid-container {
    display: grid;
    grid-template-columns: 200px 1fr;
    grid-template-rows: auto auto 1fr auto;
    gap: 20px;
    min-height: 100vh;
}
header {
    grid-column: 1 / 3;
    grid-row: 1 / 2;
    background-color: #2c3e50;
    color: white;
    padding: 20px;
}
nav {
    grid-column: 1 / 2;
    grid-row: 2 / 4;
    background-color: #f0f3f5;
    padding: 15px;
}
main {
    grid-column: 2 / 3;
    grid-row: 2 / 4;
    padding: 15px;
}
aside {
    grid-column: 1 / 3;
    grid-row: 4 / 5;
    background-color: #e8ecef;
    padding: 15px;
}
footer {
    grid-column: 1 / 3;
    grid-row: 5 / 6;
    text-align: center;
    padding: 10px;
}
  • Test with Live Server and adjust column/row sizes or gaps.

Your Homework:

  • Recreate a Blog Archive Layout: Create a new HTML/CSS file pair.
  • Design a layout with a header, a grid of article previews (e.g., 3 columns), and a footer.
  • Use Grid to arrange the article previews:
    • .articles {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
.article {
    background-color: #f9f9f9;
    padding: 15px;
    border-radius: 8px;
}
  • Add placeholder content (e.g., <article> with a title, image, and summary).
  • Ensure responsiveness by testing different browser sizes.
