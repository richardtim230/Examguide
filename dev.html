One-Month Web Development Plan: HTML & CSS

Goal: Build web pages. Structure with HTML, style with CSS. From zero to basic sites.

View JavaScript Learning Plan

Week 1: HTML - Page Structure

Theme: Web page foundations.

Day 1 (Thursday): HTML Basics & First Page

Concepts:

  • Web dev overview (client/server). HTML role.
  • Basic document structure: <!DOCTYPE html>, <html>, <head>, <body>.
  • <head> metadata (<title>, <meta charset>).
  • Block elements: <h1>-<h6>, <p>, <br>, <hr>.

Activity:

  • Create a simple bio page. Use headings, paragraphs, line breaks.

Homework:

  • Research common HTML tags.

Day 2 (Friday): Text & Lists

Concepts:

  • Text formatting: <strong>, <em>, <b>, <i>, <u>, <sup>, <sub>.
  • Lists: <ol>, <ul>, <dl>, <li>, <dt>, <dd>.
  • Links: <a> (absolute/relative, target="_blank").
  • Images: <img> (src, alt, width, height).

Activity:

  • Improve bio page: format text, add hobbies list, link to a site, add an image.

Homework:

  • Plan HTML content for a simple recipe page.

Day 3 (Saturday): Semantic HTML & Tables

Concepts:

  • Semantic HTML5: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.
  • Non-semantic: <div>, <span>.
  • Tables: <table>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>. colspan, rowspan.

Activity:

  • Re-structure bio page with semantic elements.
  • Create a simple timetable or product comparison table.

Homework:

  • Brainstorm a multi-section web page idea (e.g., portfolio, product page) using only HTML.

Week 2: CSS - Styling

Theme: Visual design.

Day 1 (Thursday): CSS Intro & Selectors

Concepts:

  • What is CSS? Separation of concerns.
  • How to include CSS: Inline, Internal (<style>), External (<link>).
  • CSS syntax (rulesets, selectors, properties, values).
  • Basic selectors: Element, Class (.class), ID (#id).
  • Cascade and Specificity (basic).

Activity:

  • Apply basic styles (text/background color) to bio page using all three methods, then refactor to external stylesheet.

Homework:

  • Experiment with colors and fonts on your recipe page using element selectors.

Day 2 (Friday): Colors, Text & Box Model

Concepts:

  • CSS Colors: Named, Hex, RGB/RGBA, HSL/HSLA.
  • Text Styling: font-family, font-size, font-weight, text-align, line-height, text-decoration, letter-spacing.
  • CSS Box Model: content, padding, border, margin.
  • box-sizing: border-box;.

Activity:

  • Style recipe page (or multi-section page) with different fonts, colors, text alignment.
  • Apply padding, borders, margins.

Homework:

  • Use Chrome DevTools to inspect box models on websites.

Day 3 (Saturday): Backgrounds & Display

Concepts:

  • Backgrounds: background-color, background-image, background-repeat, background-position, background-size.
  • Shorthand background.
  • Border properties: border-style, border-width, border-color, border-radius.
  • Display properties: block, inline, inline-block.
  • Chrome DevTools for styles.

Activity:

  • Add a background image.
  • Style borders.
  • Experiment with display on <li> to make them horizontal.

Homework:

  • Plan layout of your multi-section page.

Week 3: CSS - Layout & Positioning

Theme: Arranging elements.

Day 1 (Thursday): Floats & Basic Positioning

Concepts:

  • Floats (float: left;, float: right;).
  • Clearing floats (clear: both;, clearfix).
  • CSS Positioning: static, relative, absolute, fixed, sticky.
  • top, right, bottom, left. z-index.

Activity:

  • Create a two-column layout with floats.
  • Position a "Back to Top" button with fixed.

Homework:

  • Recreate a simple magazine layout using floats and basic positioning.

Day 2 (Friday): Flexbox - 1D Layout

Concepts:

  • Flexbox Intro (1D layout).
  • Flex Container properties: display: flex;, flex-direction, justify-content, align-items, flex-wrap, align-content.
  • Flex Item properties: order, flex-grow, flex-shrink, flex-basis, flex, align-self.

Activity:

  • Convert previous float layouts to Flexbox.
  • Create a responsive navigation bar, distribute items evenly.

Homework:

  • Build an image gallery or product cards using Flexbox.

Day 3 (Saturday): CSS Grid - 2D Layout

Concepts:

  • CSS Grid Intro (2D layout).
  • Grid Container properties: display: grid;, grid-template-columns, grid-template-rows, gap.
  • Grid Item properties: grid-column, grid-row.
  • fr unit, repeat(), minmax().

Activity:

  • Create a complex page layout (dashboard, blog post) with CSS Grid.
  • Place items in specific grid areas.

Homework:

  • Recreate a common website layout (e.g., blog archive) using CSS Grid.

Week 4: Advanced Styling & Project

Theme: Interactivity, refinement, full project.

Day 1 (Thursday): Transitions, Transforms & Basic Animations

Concepts:

  • CSS Transitions: transition-property, transition-duration, transition-timing-function, transition-delay.
  • CSS Transforms: translate(), rotate(), scale(), skew().
  • Basic CSS Animations: @keyframes, animation properties.

Activity:

  • Add transition effects to buttons/links on hover.
  • Apply a subtle transform on an image.
  • Create a simple pulsating effect.

Homework:

  • Find examples of transitions and transforms online.

Day 2 (Friday): Forms & Pseudo-Classes/Elements

Concepts:

  • Styling Forms: input, textarea, select, button.
  • Pseudo-classes: :hover, :active, :focus, :nth-child(), :first-child, :last-child.
  • Pseudo-elements: ::before, ::after.

Activity:

  • Create and style a contact form.
  • Use pseudo-classes for button appearance.
  • Add decorative elements with ::before/::after.

Homework:

  • Design and style a login/registration form.

Day 3 (Saturday): Final Project & Review

Concepts:

  • HTML & CSS review.
  • Best practices: Semantic HTML, external stylesheets, naming, basic accessibility.
  • Debugging with DevTools.

Activity:

  • Work on your multi-section page idea (portfolio, product landing, blog).
  • Apply all learned HTML/CSS for structure, styling, layout.
  • Final Output: A presentable, well-structured, styled multi-page simple website.
  • Q&A: Open session for questions and troubleshooting.
